# MarkNote — 软件技巧笔记备忘录

个人学习和工作中积累的软件技巧笔记，涵盖工业设计、图形图像、前后端开发、系统运维等多个领域。

## 目录结构

| 目录 | 内容 |
|------|------|
| **3D/** | 三维建模与参数化设计（Rhino、Grasshopper、ZBrush、Marvelous Designer） |
| **Adobe/** | Adobe 系列软件（Photoshop、Illustrator、After Effects、InDesign、XD、Substance 3D） |
| **Autodesk/** | Autodesk 系列（3ds Max、AutoCAD、SketchUp） |
| **Axure/** | Axure RP 原型设计快捷键 |
| **Blender/** | Blender 综合笔记（渲染、雕刻、几何节点、材质、动画等） |
| **C#/** | C# 编程语言基础 |
| **CMD/** | Windows 批处理脚本 |
| **Code/** | Python 实用自动化脚本（图片转换、文件整理等） |
| **Docker/** | Docker 容器化部署 |
| **Excel/** | Excel 相关笔记 |
| **ffmpeg/** | FFmpeg 音视频处理 |
| **Git/** | Git 版本控制与 SSH 配置 |
| **HTML/** | 前端开发（HTML、CSS、Vue3、布局） |
| **JavaScript/** | JavaScript 语言参考 |
| **MarkDown/** | Markdown 语法指南 |
| **Materialize/** | PBR 材质贴图生成 |
| **Photography/** | 摄影构图与 ImageMagick 图片处理 |
| **PPT/** | PowerPoint 设计技巧 |
| **Python/** | Python 全面笔记（基础、数据分析、爬虫、Flask、量化交易、异步编程等） |
| **SVN/** | SVN 版本控制 |
| **Win/** | 系统运维（Windows、Linux、macOS、VPS、NAS、Cloudflare、量化交易服务器） |

## 主要内容

- **设计建模**: Rhino 参数化建模、Grasshopper 运算器、Blender 渲染/雕刻/几何节点、Substance 3D 材质
- **图形图像**: Photoshop 精修合成、Illustrator 矢量绘图、摄影构图、PBR 材质生成
- **编程开发**: Python（数据分析/爬虫/量化/Flask）、JavaScript、C#、HTML/CSS
- **量化交易**: CCXT 加密货币接口、TA-Lib 技术分析、TradingView Pine Script
- **系统运维**: Windows/Linux/macOS 配置、Docker、VPS 运维、NAS 搭建、Cloudflare CDN
- **多媒体处理**: FFmpeg 格式转换与压缩、ImageMagick 图片处理
