- **Delete：**移除选中对象（快捷键 `Backspace`）

- **Find：**查找，（快捷键 `Ctrl + F`）

- **Set default properties**设置默认属性

  > 将当前选中对象的样式（如：填充颜色、描边、阴影、字体设置）定为“默认值”

- **Copy properties：**复制属性（快捷键 `Ctrl + Alt + C`）

  > 仅复制对象的视觉特征（颜色、透明度、投影、描边、文字排版等），而不复制对象本身。

- **Select matching layers ：**选择匹配图层（快捷键 `Ctrl + Alt + A`）

  > 技巧：在不同的 Frame 里有同名的图层（比如每个页面都有一个叫 "Header" 的背景），
  >
  > 选中其中一个后按此键，Figma 会帮你跨页面选中所有同名图层。

- **Select none：**取消选择，（快捷键 `Esc`）

- **Select inverse：**（快捷键 `Ctrl + Shift + A`）

- **Select all with：**

  > Select all with same properties：选中所有**属性完全一致**的对象（颜色、大小、特效等）
  > Select all with same fill：选中所有**填充颜色/渐变**相同的对象
  > Select all with same stroke：选中所有**描边颜色或粗细**相同的对象
  > Select all with same effect：选中所有**投影、外发光或模糊效果**一致的对象
  > Select all with same text properties：选中所有**文字样式**（字号、行高、字重等）完全一致的文本
  >
  > Select all with same font：只要**字体名称**相同就选中，不看字号
  > Select all with same instance：选中所有引用自同一个 **Component (组件)** 的实例
  > Select all with same variant：选中所有处于同一个 **Variant (变体)** 状态的组件（例如选中所有“处于禁用状态的按钮”）

###### Object (对象)

- **Frame selection：**（快捷键 `Ctrl + Alt + G`）

  > 框架化，Frame 像是一个独立的画布，拥有自己的填充、描边、裁切（Clip content）和 Auto Layout 属性
  >
  > 几乎所有 UI 元素和页面布局都应使用 Frame

- **Group selection：**（快捷键 `Ctrl + G`）

  > 编组
  >
  > 逻辑上的简单归类，组的大小由子图层决定，没有自己的背景色或裁切属性
  >
  > 移动方便，不涉及复杂布局

- **Ungroup selection：**（快捷键 `Ctrl + Shift + G`）

  > 取消编组，拆散容器，将内部对象释放到上一级

- **Wrap in new section：**（快捷键 `Ctrl + S`）——封装进新区域

  > 封装进新区域

- **Convert to section：**转换为区域

  > 比 Frame 更高一级的组织单位
  >
  > Section 用于在巨大的画布上划分“功能区”（如：量化面板区、CAD 图纸存放区）

- **Convert to frame：**

  > 允许你在不同等级的容器之间灵活切换

- **Use as mask：**（快捷键 `Ctrl + Alt + M`）

  > 作为遮罩
  >
  > 用当前形状去裁剪上方的图层，只有形状范围内的内容可见

- **Restore default thumbnail：**

  > 恢复默认缩略图

- **Add auto layout：**（快捷键 `Shift + A`）

  > 当你把一个 Frame 设为文件的封面（Thumbnail）后，可以用此项恢复

- **More layout options：**

  > 打开详细的对齐、内边距（Padding）和间距设置面板

- **Create component：**（快捷键 `Ctrl + Alt + K`）——创建组件

  > 定义一个“母版”。修改母版，所有引用的副本（Instance）都会同步更新
  >
  > 按钮、导航栏、K线图图标

- **Slots：**

  > Convert to slot：（快捷键 `Ctrl + Shift + S`）——将选中的图层或 Frame 转换为一个插槽占位符
  >
  > Convert to frame：将插槽还原为普通的 Frame 容器
  >
  > Wrap in new slot：封装进新插槽
  >
  > Reset slot：重置插槽——撤销在实例中对插槽内容的修改，恢复到组件定义的初始状态

- **Reset instance：**重置实例

  > 一键清除所有在实例上做的手动覆盖（Overrides），使其完全还原成母版的模样

- **Detach instance：**（快捷键 `Ctrl + Alt + B`）——脱离实例

  >  断开与母版的链接。使其变回普通的图层/框架，不再随母版更新
  >
  > 当你需要对某个特定按钮进行彻底的破坏性修改，而不影响全局样式时

- **Main component：**主组件/母版操作

  > Go to main component ：（快捷键 `Ctrl + Alt + Shift + K`）——跳转到主组件
  >
  > Push changes to main component：推送到主组件
  >
  > Restore main component：恢复主组件

- **Bring to front：**（快捷键`]`）——移至最前

- **Bring forward：**（快捷键`Ctrl + ]`）——前移一层

- **Send backward：**（快捷键`Ctrl + [`）——后移一层

- **Send to back：**（快捷键`[`）——移至最后

- **Flip horizontal：**（快捷键`Shift + H`）——水平翻转

- **Flip vertical：**（快捷键`Shift + V`）——垂直翻转

- **Rotate 180° :**

  > 将对象完全倒置

- **Rotate 90° left:**

  > 逆时针旋转 90 度

- **Rotate 90° right:**

  > 顺时针旋转 90 度

- **Flatten:**（快捷键`Alt + Shift + F`）——拼合路径，

  > 将多个图层或复杂的矢量图层合并为一个单一的矢量路径图层
  >
  > 完成设计后，通过拼合来减少图层数量，优化导出的 SVG 代码

- **Outline stroke:**（快捷键`Ctrl + Alt + O`）

  > 将“描边”属性转换为“形状”

- **Boolean groups:**布尔运算

  > Union：（快捷键`Alt + Shift + U`）——联集/并集
  >
  > Subtract：（快捷键`Alt + Shift + S`）——减集/差集
  >
  > Intersect：（快捷键`Alt + Shift + I`）——交集
  >
  > Exclude：（快捷键`Alt + Shift + E`）——排除/差分

  > 显示/隐藏选中项

- **Lock/Unlock selection:** （快捷键` Ctrl + Shift + L`）

  > 锁定/解锁选中项

- **Hide other layers:**隐藏其他图层

  > 快速聚焦于当前选中的元素，屏蔽视觉干扰

- **Collapse layers:** （快捷键` Alt + L`）——折叠图层

  > 一键收起所有展开的 Frame 和 Group，瞬间恢复清爽

- **Remove fill:**（快捷键` Alt + /`）——移除填充

- **Remove stroke:**（快捷键` Shift + /`）——移除描边

- **Swap fill and stroke:**（快捷键` Shift + X`）——交换填充与描边

  > 快速切换零件图的实心填充模式与线框模式

- **Remove interactions:**移除交互

  > 一键清除所有 Prototype（原型）连线，常用于从交互稿转为静态说明文档

- **Delete contents:**删除内容

  > 保持容器（Frame/Group）不变，但清空内部所有子元素

###### Vetor（矢量）

- **Join selection:**（快捷键 `Ctrl + J`）

  > 合并所选路径
  >
  > 将两个分离的端点（Anchor points）连接成一条闭合或连续的路径

- **Smooth join selection**

  > 平滑合并
  >
  > 在连接两个端点的同时，自动调整锚点的控制柄（Handles），使连接处呈现自然的弧度

- **Delete and heal selection:** （快捷键 `Shift+Backspace`）

  > 删除并修复
  >
  >  在删除某个锚点的同时，Figma 会自动重新连接相邻的锚点，并尝试保持原有路径的形状

- **Split vector:**拆分矢量

  > 将一个复杂的矢量对象拆分为独立的部分

- **Simplify vector**简化矢量

  > 在几乎不改变形状的前提下，自动删除多余的锚点

- **Offset vector**路径偏移/扩缩

  > 在原有路径的基础上，向外或向内等距离生成一个平行的轮廓线

###### Preferences （首选项）

- **Snap to geometry:**吸附到几何图形

  > 拖动时自动对齐到形状的边缘、中点或锚点

- **Snap to objects:**吸附到对象，自动对齐到画布上其他图层的边缘或中心

  > 确保移动后的坐标始终为整数，避免出现 $0.1px$ 的模糊边缘

- **Keep tool selected after use:**

  > 使用一次工具（如画矩形）后不跳回选择工具，方便连续绘制

- **Use alternate zoom handling:**

  > 改变缩放的灵敏度或逻辑（通常针对触控板用户）

- **Highlight layers on hover:**

  > 鼠标悬停在对象上时，画布上会显示蓝框，图层面板也会同步高亮

- **Rename duplicated layers:**复制图层时自动重命名

- **Show dimensions on objects:**

  > 选中对象时，直接在画布上实时显示其宽度和高度数值

- **Hide canvas UI during changes:**

  > 在移动或缩放图层时，暂时隐藏坐标线等 UI，减少视觉干扰

- **Use smart quotes/symbols**

  > 自动将直引号 `'` 转换为弯引号 `‘`，或将 `--` 转换为长破折号 `—`

- **Flip objects while resizing:**

  > 允许通过向反方向拉伸尺寸调节柄来直接翻转对象

- **Keyboard zooms into selection:**

  > 使用快捷键缩放时，自动以选中物体为中心，而不是屏幕中心

- **Invert zoom direction:**

  > 反转鼠标滚轮缩放的方向

- **Ctrl+click opens right click menus:**

  > 为 Mac 用户或习惯单键鼠标的用户设计，用 Ctrl+左键 代替右键

- **Use number keys for opacity:**

  > 开启后，按下数字键 1-9 直接改变透明度（如按 5 透明度变为 50%）

- **Use old shortcuts for outlines:**
