##### 元素定位

- 绝对定位
- 相对定位

###### position

| 值         | 脱离文档流？ | 参考点               | 常用场景             |
| :--------- | :----------- | :------------------- | :------------------- |
| `static`   | ❌            | 无（默认）           | 普通元素             |
| `relative` | ❌            | 自身原位置           | 微调；设为定位父容器 |
| `absolute` | ✅            | 最近非 `static` 祖先 | 弹窗、角标、覆盖层   |
| `fixed`    | ✅            | 浏览器视口           | 固定导航、悬浮按钮   |
| `sticky`   | ⚠️ 滚动时切换 | 滚动容器 + 阈值      | 吸顶表头、目录       |
