##### 标题与段落

```html
<h1>到<h6>：标题（h1 最大，h6 最小）
<p>：段落
<br>：换行（自闭合标签）
<hr>：水平线
```

##### a表签

```html
<a href="目标地址">链接文本</a>
```

| 类型     | 示例                            | 说明                                                   |
| :------- | :------------------------------ | :----------------------------------------------------- |
| 外部链接 | `https://www.example.com`       | 跳转到其他网站                                         |
| 内部页面 | `/about.html` 或 `contact.html` | 站内跳转                                               |
| 页面锚点 | `#section1`                     | 跳转到当前页面的某个元素（需该元素有 `id="section1"`） |
| 邮箱链接 | `mailto:example@example.com`    | 打开默认邮件客户端                                     |
| 电话链接 | `tel:+8613800138000`            | 在手机上拨打电话                                       |
| 文件下载 | `files/document.pdf`            | 下载或打开文件                                         |

###### target

| 值                 | 说明                            |
| :----------------- | :------------------------------ |
| `_self`（默认）    | 在当前窗口/标签页打开           |
| `_blank`           | 在新标签页打开（**常用**）      |
| `_parent` / `_top` | 用于框架（frame）环境，现已少用 |
