##### 软件包管理

| 命令                       | 说明                                                     |
| -------------------------- | -------------------------------------------------------- |
| `sudo apt update`          | 更新软件源列表 (修改 `/etc/apt/sources.list` 后必须运行) |
| `sudo apt upgrade`         | 升级所有已安装的软件包到最新版本                         |
| `sudo apt install <包名>`  | 安装软件 (例: `sudo apt install nginx`)                  |
| `sudo apt remove <包名>`   | 卸载软件 (保留配置文件)                                  |
| `sudo apt purge <包名>`    | 彻底卸载软件 (删除配置文件)                              |
| `sudo apt autoremove`      | 删除不再需要的依赖包 (清理垃圾)                          |
| `sudo apt search <关键词>` | 搜索软件包                                               |
| `apt show <包名>`          | 查看软件包详细信息                                       |
| `dpkg -i <文件名.deb>`     | 安装本地下载的 `.deb` 包                                 |

##### 系统管理与服务控制

| 命令                              | 说明                                      |
| --------------------------------- | ----------------------------------------- |
| `systemctl status <服务名>`       | 查看服务状态 (例: `systemctl status ssh`) |
| `sudo systemctl start <服务名>`   | 启动服务                                  |
| `sudo systemctl stop <服务名>`    | 停止服务                                  |
| `sudo systemctl restart <服务名>` | 重启服务                                  |
| `sudo systemctl enable <服务名>`  | 设置开机自启                              |
| `sudo systemctl disable <服务名>` | 禁止开机自启                              |
| `sudo reboot`                     | 重启系统                                  |
| `sudo shutdown -h now`            | 立即关机                                  |
| `hostnamectl`                     | 查看或修改主机名                          |
| `timedatectl`                     | 查看或修改系统时间/时区                   |

##### 用户与权限管理

| 命令               | 说明                                         |
| ------------------ | -------------------------------------------- |
| `sudo <命令>`      | 以超级管理员 (root) 权限执行命令             |
| `su -`             | 切换到 root 用户 (需要输入 root 密码)        |
| `adduser <用户名>` | 创建新用户 (Debian 推荐的高级脚本，交互友好) |
| `deluser <用户名>` | 删除用户                                     |
| `passwd <用户名>`  | 修改用户密码                                 |
| `chmod`            | 修改文件权限                                 |
| `chown`            | 修改文件所有者                               |
| `whoami`           | 显示当前用户                                 |
| `id`               | 显示用户 UID/GID 及所属组                    |

##### 网络与防火墙

| 命令                           | 说明                                       |
| ------------------------------ | ------------------------------------------ |
| `ip addr` (或 `ip a`)          | 查看 IP 地址 (替代旧的 `ifconfig`)         |
| `ip route`                     | 查看路由表 (替代旧的 `route`)              |
| `ss -tulpn`                    | 查看监听端口和进程 (替代旧的 `netstat`)    |
| `ping <IP或域名>`              | 测试网络连通性                             |
| `curl` / `wget`                | 下载文件或测试 HTTP 请求                   |
| `sudo ufw status`              | 查看 UFW 防火墙状态 (如果已安装)           |
| `sudo nft list ruleset`        | 查看 Nftables 防火墙规则 (Debian 默认底层) |
| `nano /etc/network/interfaces` | 编辑传统网络配置文件 (Debian 特色)         |
| `nmcli`                        | 如果使用 NetworkManager，用此命令管理网络  |

##### 进程与资源监控

| 命令            | 说明                                                        |
| --------------- | ----------------------------------------------------------- |
| `top`           | 实时查看进程和资源占用 (按 `q` 退出)                        |
| `htop`          | `top` 的增强版 (需安装 `sudo apt install htop`)，界面更友好 |
| `ps aux`        | 静态查看所有进程快照                                        |
| `kill <PID>`    | 终止进程 (发送 SIGTERM)                                     |
| `kill -9 <PID>` | 强制杀死进程 (发送 SIGKILL)                                 |
| `free -h`       | 查看内存使用情况 (人类可读格式)                             |
| `df -h`         | 查看磁盘空间使用情况                                        |
| `du -sh <目录>` | 查看特定目录的大小                                          |
